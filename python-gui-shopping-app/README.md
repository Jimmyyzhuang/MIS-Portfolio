# Shopping Application GUI

## Overview
This project is a Python GUI shopping application built using appJar.
It allows users to browse items by category and make selections through
a graphical interface.

## Features
- GUI built with appJar
- Category selection (Jerseys, Hats, Accessories)
- Separate chatbot logic file
- Reads product data from a CSV file

## How to Run
1. Install dependencies:
   pip install pandas appJar
2. Run the program:
   python codetemplate.py

## Files
- codetemplate.py: Main GUI application
- chatbot.py: Chatbot logic
- primethreads.csv: Product data
- PrimeThreads.gif: GUI image asset
